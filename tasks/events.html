Events - Overview

Events for event handling for clicks, scrolls or drags (..etc)

Example number #01
events_01.html 
<!-- HTML - onclick event with function (named here)- doClick -->
<!-- Notice that the button tag is unstyled -->
<button onclick="doClick()">Click Me</button>
events_01.js 
function doClick(){
  alert('You clicked!');
}
